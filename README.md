# AIO Life

> 记录、统计、分析人生的所有数据
>
> **AIO Life** 是一款 All-in-One 人生管理系统，致力于全方位记录、统计与分析人生的所有数据。通过主动录入与第三方数据拉取，实现人生痕迹的全面数字化，助您洞察数据，掌控生活节奏。

# 项目背景

个人比较喜欢看各种数据统计，希望能够通过一个系统量化自己的人生数据。也希望通过数据的支撑，更好地驱动和掌控自己的生活节奏。


# 效果

- 用上了这个系统之后，因为在首页设置了四个每日卡片，除了一些意外的情况，基本上每天都可以完成卡片上的事项（比如坚持运动，Leetcode 每日打卡，Github持续提交等）。

- 因为会记录每一分钟时间的花费，也会尽可能的避免浪费时间和频繁切换任务（切换任务会导致多次记录，因为懒得多次记录，就尽可能一次性完成任务）。


## 功能

### 主页

动态化、可配置的数据看板，五张每日卡片由后端下发，支持以下类型：

- **时迹**：当日时间投入与当前状态
- **今日运动**：当日运动记录汇总
- **GitHub**：调用 GitHub 接口查询提交情况
- **LeetCode**：每日一题完成情况，点击跳转到对应题目
- **扇贝单词**：调用扇贝单词接口查询打卡状态

此外主页还包含：时迹统计面板、关注的待办任务、固定闪念、常用功能快捷导航，长按时迹 / 运动卡片可一键录入。

### 任务中心

- **待办**：跟踪待办事项，支持优先级、关注、完成时间
- **目标管理**：长期目标的制定与跟踪

### 时间管理

- **时迹**：精确记录每一分钟的时间花费，任务完成后即时录入；提供列表 / 卡片两种视图
- **看板**：时间投入的可视化统计
- **我的分类**：个人时间分类管理
- **分类管理**（管理员）：全局时间分类维护

### 生活记录

- **运动**：日常运动记录与分类配置，含每日统计图表
- **视频观看**：通过浏览器插件在 B 站观看时自动同步指定视频进度
- **闪念**：随时记录灵感与想法，支持置顶展示在主页
- **笔记**：随手记录文章与备忘
- **活动**：记录演出、展览等线下事件
- **里程碑**：记录生活与成长的关键节点
- **纪念日**：记录重要日期
- **荣誉中心**：记录获得的成就与荣誉

### 财务中心

- **概览**：收入、支出与结余看板
- **收入**：记录每一笔收入
- **支出**：记录每一笔支出，移动端适配
- **账单导入**：支持支付宝和微信账单导入，手机支付宝账单可自动匹配分类

### 物品中心

- **设备墙**：记录电子设备等
- **衣柜**：记录衣服、裤子和鞋子等，支持购买日期等属性

### 关系图谱

以可视化图谱展示人际关系网络，支持节点交互浏览。

### 编程看板

- **GitHub 看板**：提交记录与统计
- **LeetCode 看板**：刷题记录与统计
- **CSDN 看板**：博客数据统计

### 密码管理

集中管理各类账号密码，支持密码安全存储与编辑。

### AI 与 MCP

- **LLM 设置**：大模型相关配置
- **MCP 工具**：模型上下文协议工具调用，支持模拟调用调试

### 个人中心

- **密码修改**：修改登录密码
- **头像修改**：上传与更换头像
- **第三方账号绑定**：GitHub、LeetCode、扇贝等

### 消息中心

- **邮件通知**：通过邮件接收系统消息
- **站内通知**：站内信通知

### 系统管理

- **用户中心**：查看与管理系统用户
- **权限菜单**：管理系统菜单与权限
- **用户字典管理**：管理用户自定义的字典数据（如支出分类、支付方式等）

### 配置管理

> 权限隔离尚未完成，目前仅管理员可访问

- **字典类型**：维护系统全局字典类型
- **字典数据**：维护系统全局字典数据

## 快速开始

### 环境准备

- Node.js >= 20.10.0
- pnpm >= 9.12.0

### 安装依赖

```bash
# 启用 corepack (如果尚未启用)
corepack enable

# 安装依赖
pnpm install
```

### 启动项目

```bash
# 启动应用
pnpm run dev
```

### 构建项目

```bash
# 构建应用
pnpm run build
```
